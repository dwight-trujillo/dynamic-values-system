Dynamic Values System - Technical Documentation v4.0

Architecture Overview

The system implements a hexagonal architecture (Ports & Adapters pattern) for maximum flexibility and testability.

Core Components

  • ValueManager - Central orchestrator for all dynamic values
  • CacheInterface - Redis cluster implementation with automatic failover
  • DataSources - Pluggable data sources (Random, Database, API, WebSocket)
  • Security Layer - WAF, Rate Limiting, Input Validation

API Endpoints

  • GET /api.php - Returns all dynamic values
  • GET /api.php?key={key} - Returns a specific value by key
  • GET /health.php - Health check endpoint

Security Features

  • WAF with OWASP Core Rule Set
  • Rate limiting: 100 requests per minute per IP
  • Input validation and XSS sanitization
  • Security headers (CSP, HSTS, X-Frame-Options)

Deployment

docker-compose up -d
curl http://localhost:8080/api.php
